Travel agency:
- Design and implement a client-server application for a travel agency. 
- 2 types of user: client and agent
The client will be able to do the following:
	- Application registration
	- View the curently available bids by filtering destinations
	- Selection and booking a package (application must display available places including period)
	- The ability to use a voucher
	- Add review to a specific package booked
The agent can do:
	- CRUD users
	- CRUD packages
	- PDF format reports for sales
	- Make promotions

The application will be web using Java - Sprin and MySQL databeses
